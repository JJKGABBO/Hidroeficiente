  • Riego Automático
  • Sistema tecnológico que suministra agua a los cultivos de manera autónoma, utilizando sensores y controladores para ajustar el flujo de agua según las necesidades específicas de las plantas.
  • Un sistema que riega un cultivo de tomates cada vez que los sensores detectan que la humedad del suelo está por debajo del 30%
    • Microcontrolador
    • Circuito integrado programable que funciona como el "cerebro" de un sistema de riego automatizado, procesando información de sensores, tomando decisiones y controlando actuadores.
  • Arduino Uno controlando las válvulas de riego de un invernadero, activándolas y desactivándolas según los datos de los sensores.
    • Sensor de humedad del suelo
    • Dispositivo que mide el contenido de agua en el suelo, proporcionando datos precisos para determinar cuándo es necesario regar.
  • Ejemplo: Un sensor capacitivo insertado en un campo de maíz que indica cuándo el suelo está demasiado seco y necesita riego.
    • Electroválvula
    • Dispositivo electromecánico que controla el flujo de agua mediante la apertura y cierre automático, activado por señales eléctricas.
  • Una válvula que se abre automáticamente a las 6 am para regar un jardín de rosales y se cierra después de 20 minutos.
    • Internet de las Cosas (IoT)
    • Tecnología que permite la monitorización y control remoto de sistemas de riego a través de redes de sensores inalámbricos y aplicaciones móviles.
  • Una aplicación móvil que permite a un agricultor monitorear y controlar el sistema de riego de sus cultivos desde su teléfono inteligente.
    • Eficiencia Hídrica
    • Principio que busca utilizar el agua de manera óptima, suministrando únicamente la cantidad necesaria para los cultivos y minimizando el desperdicio.
  • Reducir el consumo de agua en un huerto de manzanos de 100 litros por día a 60 litros, manteniendo la misma productividad.
    • Sensores de temperatura
    • Dispositivos que miden la temperatura ambiente para ayudar a ajustar los ciclos de riego según las condiciones climáticas.
  • Un sensor que detecta que la temperatura ambiente ha subido a 35°C y aumenta la frecuencia de riego para compensar la mayor evaporación.
    • Electroválvula Solenoides
    • Tipo específico de válvula que utiliza un campo electromagnético para controlar el flujo de líquidos, permitiendo la apertura y cierre
  • Una válvula que se abre instantáneamente cuando recibe una señal eléctrica de 12V para iniciar el riego de una sección de un cultivo de pepinos.
    • Agricultura de precisión
    • Enfoque agrícola que utiliza tecnologías avanzadas para optimizar el uso de recursos, mejorando la productividad y sostenibilidad.
  • Usar drones con sensores multiespectrales para identificar exactamente qué partes de un campo de trigo necesitan m
    • Balance hídrico del suelo
    • Relación entre la entrada de agua (lluvia o riego) y la salida de agua (evapotranspiración, percolación) en un terreno agrícola.
    Calcular que en un mes de cultivo de arroz, entran 500 mm de agua por lluvia y riego, y salen 450 mm por evapotranspiración.

    Glosario Pensamiento Algoritmico

    • Algoritmo
    • Una secuencia de pasos bien definidos para resolver un problema o realizar una tarea. Es como una receta, pero para resolver problemas computacionales.
    Instrucciones para armar un mueble: La secuencia de pasos para ensamblar las piezas es un algoritmo.
    • Pensamiento Algoritmico
    • La habilidad de descomponer problemas en pasos más pequeños y de diseñar secuencias lógicas para resolverlos. Es la base de la programación y la resolución de problemas en general.
    Resolver un rompecabezas: Descomponer el problema en pasos más pequeños para encontrar la solución.
    • programacion
    • El proceso de escribir código para crear programas que ejecutan algoritmos. Los lenguajes de programación (como Python, Java, C++) son herramientas para expresar estos algoritmos.
  • Crear una aplicación móvil: Escribir código en un lenguaje de programación para definir la interfaz de usuario, la lógica y las funcionalidades de la app.
    • Estructura de datos
    • Formas organizadas de almacenar y gestionar datos. Ejemplos incluyen listas, arrays, árboles y grafos.
  • Árbol genealógico: Una estructura jerárquica que representa las relaciones familiares.
    • Complejidad Algoritmica
    • Mide la eficiencia de un algoritmo en términos de tiempo y espacio utilizados. Es importante para elegir el algoritmo más adecuado para una tarea.
  • Buscar un elemento en una lista desordenada: Un algoritmo de búsqueda lineal tiene una complejidad mayor que un algoritmo de búsqueda binaria en una lista ordenada.
    • Sistema
    • Un conjunto de componentes interrelacionados que trabajan juntos para lograr un objetivo común. Los sistemas pueden ser físicos (como una computadora) o abstractos (como un sistema operativo).
  • Una computadora: Un sistema compuesto por hardware y software que trabajan juntos.
    • Sistea Operativo
    • Software que gestiona los recursos de hardware de una computadora y proporciona una interfaz para que los usuarios puedan interactuar con el sistema.
  • Windows, macOS, Linux: Sistemas operativos que gestionan los recursos de una computadora y proporcionan una interfaz de usuario.
    • Software
    • Programas de computadora que realizan tareas específicas. Pueden ser aplicaciones, sistemas operativos, utilidades, etc.
  • Un juego de video: Un programa que permite a los usuarios interactuar en un mundo virtual.
    • Hardware
    • Los componentes físicos de una computadora, como la CPU, la memoria, los discos duros y los dispositivos periféricos.
  • Memoria RAM: Almacena datos temporalmente mientras la computadora está en funcionamiento.
    • Inteligencia Artificial
    • El desarrollo de sistemas informáticos capaces de realizar tareas que normalmente requieren inteligencia humana, como el aprendizaje,1 el razonamiento y la resolución de problemas.2  
    • Asistentes virtuales: Como Siri o Alexa, capaces de responder preguntas y realizar tareas.
